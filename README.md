# Python File & Folder Manager

A simple Command Line Interface (CLI) project built using Python that allows users to manage files and folders.

## Features
- Create folders
- View files and folders
- Rename folders
- Delete folders
- Create files
- Read file content
- Update file (rename, append, overwrite)
- Delete files

## Technologies Used
- Python
- pathlib module
- os module

## How It Works
The program runs in a loop and provides options for performing different file and folder operations. Users can choose an option from the menu and perform actions directly from the terminal.

## Learning Outcome
This project helped me understand:
- File handling in Python
- Exception handling
- Using pathlib for file system operations
- Building CLI-based applications
